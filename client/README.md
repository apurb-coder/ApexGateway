# ApexGateway - Client Web Application

React 19 + Vite + TailwindCSS v4 frontend application for ApexGateway.

For complete project documentation, system architecture, environment setup, and API specifications, please refer to the [ApexGateway Root README](../README.md).

## Quick Start

```bash
# Install dependencies
npm install

# Start Vite dev server
npm run dev
```

The application will run locally at `http://localhost:5173`.
